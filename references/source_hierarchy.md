# 数据来源和搜索规则

## 来源优先级

1. 监管文件：FDA review、EMA EPAR、NMPA 审评文件、药品标签、说明书。
2. 临床注册结果：ClinicalTrials.gov、EU CTIS、中国药物临床试验登记、JPRN、ISRCTN 等。
3. 同行评议论文和补充材料。
4. 大会摘要、海报和口头报告：AHA、ACC、ESC、ASCO、ASH、ESMO、EULAR、ERA、ASN 等。
5. 公司官网、新闻稿、投资者 PPT、年报、招股书。
6. 第三方数据库或新闻，仅作线索；除非没有更高优先级来源，不作为关键数字唯一来源。

## 通用搜索

每条管线至少搜索：

- 药物名 + 公司 + 适应症。
- 药物名 + target/mechanism + indication。
- 药物名 + trial number / NCT。
- 药物名 + phase + data + safety。
- 药物名 + conference abstract/poster。
- 公司名 + 药物名 + press release / pipeline。

## 已上市药物

已上市药物要搜索：

- 标签/说明书和标签安全性。
- 相关适应症下所有重要注册临床。
- 与目标管线最相关的机制、影像、亚组或终点研究。
- 上市后重大安全性警示。

若上市药没有斑块或目标终点数据，仍可作为商业/标准治疗标尺，但在 `临床实验的可比性` 中说明“非直接斑块管理研究”。

## 未上市药物

未上市药物要搜索：

- 所有公开临床登记。
- 临床前论文、专利线索、会议摘要、公司新闻稿。
- 1 期/2 期/3 期结果披露。
- 终止、撤回、长期无更新或资产转让信息。

只登记但未披露结果时，写“仅登记，未披露结果”。

## 斑块/影像研究搜索词

动脉粥样硬化、冠脉斑块、斑块管理项目额外搜索：

- atherosclerosis plaque volume。
- PAV / percent atheroma volume。
- TAV / total atheroma volume / normalized TAV。
- IVUS / CCTA / OCT / NIRS。
- NCPV / low attenuation plaque / LAPV / LCBI / fibrous cap thickness。
- CIMT / IMT。
- plaque regression / plaque stabilization / MACE。

## 记录口径

每个关键数字必须能追溯来源。原始数据池中记录来源质量和链接，量化横比中保留来源行。

若同一数字来自多个来源：

- 优先记录最高等级来源。
- 其他来源可放备注。
- 数值不一致时说明差异，不能静默合并。

## 未披露处理

没有公开数据时写：

- 未披露。
- 不适用。
- 仅登记，未披露结果。
- 仅定性披露，未披露量化数据。
- 基线未披露，无法计算比例变化。

不要用推测、同类药物或图形估读填补。若确需估读或估算，必须标注“估读”或“按基线估算”。
