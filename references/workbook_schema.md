# Excel 工作簿结构

## 默认 Sheet 顺序

1. `{管线}的可比管线`
2. `{管线}_{适应症}`
3. `{管线}_{适应症}_量化横比`

可选 sheet：

- `指标口径`
- `{管线}_{适应症}_披露总览`
- `{管线}_{适应症}_量化横比_精简版`

## `{管线}的可比管线` 字段

- 序号
- 管线/药物
- 所属公司/机构
- 管线可比性归类
- 管线可比性归类说明
- 管线可比性量化评分（10分）
- 命中可比维度
- 靶点/载荷
- 机制
- 技术形式/给药
- 适应症/目标人群
- 最高临床阶段/商业状态
- 所属公司/机构是否有在推进
- 是否有在推进的原因
- 首次进入临床时间
- 最近公开进展时间
- 与目标管线的核心差异
- 数据源质量
- 数据源链接
- 备注/待核查点

`命中可比维度` 用 `/` 分隔，格式示例：

`相同靶向 / 不同载荷 / 相同技术 / 相同适应症 / 部分相同终点 / 不同阶段`

建议标色：

- `相同`：绿色。
- `部分相同`：黄色。
- `不同`：红色。
- `不适用`：灰色。

## `{管线}_{适应症}` 原始数据池字段

- 序号
- 管线/药物
- 所属公司/机构
- 管线可比性归类
- 管线可比性量化评分（10分）
- 命中可比维度
- 靶点/载荷
- 机制
- 技术形式/给药
- 适应症/目标人群
- 最高临床阶段/商业状态
- 所属公司/机构是否有在推进
- 是否有在推进的原因
- 与目标管线的核心差异
- 研究/数据序号
- 临床实验的可比性
- 临床研究编号/研究名称
- 临床期/数据类型
- 进入该期临床的时间点
- 适应症/模型/入组
- 主要终点
- 次要终点
- 基线
- 有效性（分队列/整体，最新披露）
- 安全性（分队列/整体，最新披露）
- 上市/标签安全性或黑框
- 数据源质量和来源小结
- 数据源链接
- 备注/待核查点

## `{管线}_{适应症}_量化横比`

固定前两列：

- 模块
- 行标题

后续每列是一条原始数据池中的研究/数据行。列名建议：

`{序号}-{研究/数据序号} {管线/药物}｜{临床研究编号/研究名称}`

推荐行块：

### 基础信息

- 管线/药物
- 所属公司/机构
- 管线可比性归类
- 管线可比性量化评分
- 命中可比维度
- 最高临床阶段/商业状态
- 所属公司/机构是否有在推进
- 临床实验的可比性
- 临床研究编号/研究名称
- 临床期/数据类型
- 进入该期临床时间
- 适应症/模型/入组
- 主要终点
- 次要终点
- 样本量/分组
- 年龄/性别
- 疾病/风险基线
- 关键生物标志物基线
- 影像/病理基线

### 疗效

按适应症拆分。斑块管理场景至少包括：

- PAV变化（绝对值）
- PAV变化（比例）
- TAV/N-TAV变化（绝对值）
- TAV/N-TAV变化（比例）
- NCPV/LAPV/LAP/NCP变化（绝对值）
- NCPV/LAPV/LAP/NCP变化（比例）
- CIMT/IMT变化
- 纤维帽/FCT/脂质弧/LCBI变化
- LDL-C/PCSK9/TG/炎症变化
- 事件终点/MACE
- 动物斑块面积/体积/病理
- 疗效整体结论

### 安全性

- AE/TEAE/常见不良事件
- TRAE
- SAE/死亡/停药
- 重点安全信号
- 上市/标签安全性或黑框

### 备注和来源

- 数据源质量和来源小结
- 数据源链接
- 备注/待核查点

## 格式建议

- 冻结首行和前两列。
- 表头深蓝底白字。
- 可比性归类使用不同底色。
- `命中可比维度` 中“相同/部分相同/不同/不适用”用不同颜色强调。
- 长文本自动换行。
- 量化横比列宽保持紧凑，原始数据池可更宽。
