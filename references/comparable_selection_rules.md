# 可比管线筛选规则

## 可比优先级

1. 同适应症 + 同靶向/载荷 + 同技术形式 + 同疾病修饰终点。
2. 同适应症 + 同机制/同通路 + 同关键终点，但技术形式不同。
3. 同适应症 + 同终点标尺，机制不同，用作疗效和安全性参照。
4. 同技术平台或同递送平台 + 相近疾病部位/病理机制。
5. 同靶向或同机制的学术/临床前项目，用于靶点验证。
6. 同适应症标准治疗、已上市药物或商业替代方案，只保留代表性项目。
7. 国内同适应症/同市场项目，用于市场和商业化竞争参考。

## 可比性归类

可根据项目调整名称，但必须让初学者能读懂高低关系：

- `S0 目标管线/定位参考`：目标产品本身，不参与竞争排序。
- `A 核心可比/直接对标`：适应症、机制或技术、关键终点高度接近，是最应该放在一起看的项目。
- `B 终点标尺/疗效参照`：机制不同，但同适应症且有关键临床或影像终点，可作为疗效、安全性或审评标尺。
- `C 平台或递送技术对标`：技术平台、递送方式或病灶靶向方式相近，但适应症、载荷或终点不完全一致。
- `D 同通路/同病理机制`：疾病生物学相关，但产品形式或临床定位不同。
- `E 广义商业替代/标准治疗`：同一疾病市场中可能被医生或支付方拿来比较的上市或后期产品。
- `F 靶点验证/学术项目`：多为临床前或非商业项目，用于判断机制是否成立。
- `G 国内同适应症/市场对标`：国内市场相关，但机制、技术或终点未必高度一致。

## 量化评分建议

用 10 分制，按投资人阅读价值而不是单一靶点评分：

- 同适应症/同治疗场景：0-2 分。
- 同靶向/载荷/作用部位：0-1.5 分。
- 同技术形式/递送平台：0-1.5 分。
- 同机制/通路：0-1 分。
- 同人群或模型：0-1 分。
- 同关键终点或可直接横比终点：0-2 分。
- 数据成熟度和披露质量：0-1 分。

评分要能解释。若评分 ≥8，通常应进入第一张可比管线表的靠前位置；若评分 <5，只在有商业或机制参考价值时保留。

## 命中可比维度

统一用短语表达，不写长段话：

- `相同靶向`
- `不同载荷`
- `相同技术`
- `部分相同机制`
- `相同适应症`
- `不同人群`
- `相同终点`
- `不同阶段`
- `不适用商业状态`

Excel 中建议对关键词标色：

- `相同` 绿色。
- `部分相同` 黄色。
- `不同` 红色。
- `不适用` 灰色。

## 数量控制

每条目标管线通常不超过 20 条可比。高可比项目不足时，不要强行补齐；但若用户提供截图或竞品清单，必须逐项解释为何纳入或剔除。

## 早期和后期项目

目标管线处于临床前或 1 期时，优先找同平台、同机制、同动物模型或早期 PD 数据。

目标管线已有患者疗效数据时，必须纳入后期或上市药物作为终点标尺；同一药物的不同人群、不同研究或不同剂量不要合并。
