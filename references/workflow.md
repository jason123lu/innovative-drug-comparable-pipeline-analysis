# 完整工作流

## 1. 定义目标管线

先把用户提供的信息整理成一句研究定义：

`目标管线 = 公司 + 管线 + 靶向/载荷 + 机制 + 技术形式 + 适应症/人群 + 当前阶段 + 需要回答的投资问题`

若用户只提供官网、管线描述或截图，先从官网、临床登记和公司资料补齐目标管线基础属性。

## 2. 建立可比维度

不要只用靶点做可比。至少考虑：

- 靶向或作用位点：如受体、细胞、组织、通路、疾病部位。
- 载荷或药效成分：如小分子、siRNA、抗体、脂质体、纳米颗粒、细胞因子抑制剂。
- 技术形式：如主动靶向纳米递送、LNP、ASO、mAb、基因编辑、传统小分子。
- 机制：如胆固醇外排、HDL/RCT、抗炎、降脂、斑块稳定、斑块消退。
- 适应症和人群：疾病、严重程度、是否已有斑块、是否 ACS/ASCVD/高危人群。
- 终点：是否有与目标管线相同的注册或影像终点。
- 阶段和数据成熟度：是否商业化、3期、2期、1期、临床前或仅学术验证。

## 3. 搜索并分类可比管线

先找“同适应症 + 同机制/同平台 + 同终点”的最高可比项目，再向外扩展到部分维度可比项目。通常控制在 20 条以内；如果高可比项目少，不要为了数量加入低相关管线。

对用户截图、PPT 或手工列出的项目，要逐条与自己搜索结果比对：

- 截图有但 Excel 没放的项目。
- Excel 有但截图没有的项目。
- 可比性更强或更弱的原因。
- 是否应该补入第一张可比管线表。

## 4. 生成 `{管线}的可比管线`

这是投资人的入口表。必须按可比性从高到低排序，并包含：

- 管线/药物、所属公司/机构。
- 可比性归类、归类说明、量化评分。
- 命中可比维度，统一写成 `相同靶向 / 不同载荷 / 相同技术 / 相同适应症` 这种格式。
- 靶点/载荷、机制、技术形式、适应症/人群。
- 最高临床阶段/商业状态。
- 所属公司/机构是否有在推进，以及判断原因。
- 与目标管线的核心差异。
- 来源质量、来源链接和待核查点。

## 5. 判断是否仍在推进

不要只看“最高临床阶段”。结合以下证据判断：

- ClinicalTrials.gov、EU CTIS、中国临床试验登记等是否仍为 recruiting、active、ongoing、not yet recruiting。
- 最近一次公司官网、年报、投资者材料、新闻稿是否仍列入管线。
- 距离首次进入当前阶段的时间是否异常长。
- 是否有终止、撤回、长期无更新、公司倒闭、资产转让或适应症转向。
- 已上市药物按商业化和标签状态判断，不写成“临床仍在推进”。

建议口径：

- `是`：24 个月内有官方进展、仍在招募/进行、仍列入管线或已上市销售。
- `可能是`：登记仍 active，但缺少近期公司披露。
- `存疑`：超过 36 个月无阶段推进或结果披露。
- `否`：终止、撤回、公司不再列示或明确停止。

## 6. 建研究级原始数据池

`{管线}_{适应症}` 是核心数据池。一个临床研究、一条临床前研究、一条标签安全性或一个注册终点研究各占一行。不要把同一药物的多个研究塞进一行。

排序规则：

- 先按管线在 `{管线}的可比管线` 中的可比性排序。
- 同一管线内，按 `临床实验的可比性` 从高到低排序。
- 斑块/疾病逆转相关研究优先于单纯降脂、单纯事件终点或单纯安全性研究。

## 7. 抽取数据

每个研究行都要尽量写清：

- 临床研究编号/研究名称。
- 临床期或数据类型。
- 进入该期临床时间点。
- 适应症/模型/入组。
- 主要终点和次要终点。
- 基线：主要终点和关键次要终点相关基线必须摘录到单元格。
- 有效性：分整体、剂量、队列、对照组；能算比例时写比例。
- 安全性：AE/TEAE/TRAE/SAE、死亡、停药、特别关注风险。
- 上市标签安全性或黑框。
- 数据源质量和来源链接。

临床前数据必须写明动物种属/模型、疗效和安全性的实验口径。

## 8. 建量化横比

`{管线}_{适应症}_量化横比` 从原始数据池生成。列为“同一管线的不同临床研究/数据行”，不是简单按药物分列。

行分四块：

- 基础信息：研究编号、临床期、进入时间、适应症、终点、重要基线。
- 疗效：不同疗效指标拆成不同行。
- 安全性：不同安全性指标拆成不同行。
- 备注和来源：来源质量、链接、待核查点。

如果一个指标未披露，写 `未披露/不适用`，不要把其他指标文本串到该行。

## 9. 校验

交付前至少校验：

- 高可比管线是否遗漏。
- 可比性分类、评分和命中维度是否一致。
- 已上市药物是否覆盖相关注册临床和标签安全性。
- 未上市药物是否覆盖所有公开临床研究和临床前关键披露。
- `基线`、`有效性`、`安全性` 和 `数据源链接` 是否为空。
- 量化横比是否每个研究一列、每个指标一行。
- Excel 是否存在公式错误或明显格式问题。
